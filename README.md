# Advanced Calculator 
A simple and modern calculator built with **HTML, CSS, and JavaScript**.  
It supports addition, subtraction, multiplication, division, and percentage.


## Features
- Clean, interactive buttons with hover effects  
- Neon/glow UI for a modern look  
- Responsive design for desktop and mobile
